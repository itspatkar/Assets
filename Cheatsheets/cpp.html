# Input & Output

• Output :

cout<< is used for printing output.
#include<iostream>
using namespace std;

int main(){
    cout << "Hello World!";

    return 0;
}

• Input :

cin>> is used for taking user input.
#include<iostream>
using namespace std;

int main(){
    int n;
    cout << "Enter Number:";
    cin >> n;

    return 0;
}

# Comments

• Single-line Comment :

// is used for single-line comment.
#include<iostream>
using namespace std;

int main(){
    cout << "Hello World!";

    // This is single-line comment.

    return 0;
}

• Multi-line Comment :

/* */ is used for multi-line comment.
#include<iostream>
using namespace std;

int main(){
    cout << "Hello World!";

    /* This is
    multi-line comment. */

    return 0;
}

• Constant Keyword/Variable :

const keyword is used for declaring constant (read-only) variables.
#include<iostream>
using namespace std;

int main(){
    const int x = 10;
    const string name = "Mandar"; 

    return 0;
}

• sizeof() Operator

Returns the size of a variable.
#include<iostream>
using namespace std;

int main(){
    int x = 15;
    cout << sizeof(x);

    return 0;
}

• Ternary Operator

[condition] ? [expression_1] : [expression_2]
If condition is true, then expression 1 is evaluated otherwise expression 2 is evaluated.
#include<iostream>
using namespace std;

int main(){
    int a=5, b=10, large;
    large = a > b ? a : b;
    cout << " Largest number is " << large;

    return 0;
}
