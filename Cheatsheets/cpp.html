• Local & Global Variables :

Local variables are declared inside a function or block. They can be used only by statements that are inside that function or block of code.
Global variables are defined outside of all the functions. The global variables will hold their value throughout the life-time of your program. A global variable can be accessed by any function.
Ex:
#include<iostream>
using namespace std;

// Global variables -
float f=5.5;
int n=0;

int main(){
    // Local variables -
    int x, y=10;
    float a,b;
    a = b = 2.5;

    return 0;
}

# Storage Classes

Storage class defines the scope (visibility) and life-time of variables and/or functions within a C++ Program.
Syntax:
class data_type var_name
Class Description
auto It is the default storage class for all local variables.
register It is used to define local variables that should be stored in register instead of RAM. The register should only be used for variables that require quick access such as counters.
static It instructs the compiler to keep local variable in existence during the life-time of the program instead of creating and destroying it each time it comes into and goes out of scope.
extern It is used to give reference of a global variable that is visible to ALL the program files.
mutable The mutable specifier applies only to class objects. It allows a member of an object to override const member function. That is, a mutable member can be modified by a const member function.
Ex:
auto int x;
register int amount = 0;
static int count = 1;

# Decision Making

• if statement :

Syntax:
if(condition){
    // statements if true
}
Ex:
int a = 10, b =20;

if(a > b){
    cout << "A is larger than B";
}

• if-else statement :

Syntax:
if(condition){
    // statements if true
} else {
    // statements if false
}
Ex:
int a = 10, b =20;

if(a > b){
    cout << "A is greater than B";
} else {
    cout << "A is smaller than B";
}

• else-if statement :

Syntax:
if(condition){
    // statements if true
} else if (condition) {
    // statements if true
} else {
    // statements if false
}
Ex:
int a = 10, b =20;

if(a > b){
    cout << "A is greater than B";
} else if (a < b) {
    cout << "A is smaller than B";
} else {
    cout << "A is equal to B";
}

• Nested if & else :

If statements inside another if or else statements.
Ex:
int age = 16;

if(age > 0){
    if(age < 18)
        cout << "You are young.";
    else if ( age >= 18)
        cout << "You are adult.";
} else {
    if(age < 0)
        cout << "Age cannot be negative.";
    else
        cout << "Age cannot be zero.";
}

• Switch Case :

Switch statement allows a variable to be tested for equality against a list of values.
Syntax:
switch(expression){
    case constant-expression-1 :
        statement(s);
        break;
    case constant-expression-2 :
        statement(s);
        break;
    case constant-expression-N :
        statement(s);
        break;

    default :
        statement(s);
}
Ex:
int day = 3;

switch(day){
    case 1 :
        cout << "Mon";
        break;
    case 2 :
        cout << "Tue";
        break;
    case 3 :
        cout << "Wed";
        break;
    case 4 :
        cout << "Thu";
        break;
    case 5 :
        cout << "Fri";
        break;
    case 6 :
        cout << "Sat";
        break;
    case 7 :
        cout << "Sun";
        break;

    default :
        cout << "Incorrect Day!";
}

# Loops

A loop statements repeatedly executes a target statement as long as a given condition is true.

• while loop :

Syntax:
while(condition) {
    statement(s);
}
Ex:
int n = 1;
while(n <= 5){
    cout << "Hello World!";
    n++;
}

• do-while loop :

Syntax:
do {
    statement(s);
} while(condition);
Ex:
int n = 1;
do {
    cout << "Hello World!";
    n++;
} while(n <= 5)

• for loop :

Syntax:
for ( init; condition; increment ){
    statement(s);
}
Ex:
for(int i=1; i<=5; i++){
    cout << "Hello World!";
}

• Nested loop :

A loop can be nested inside of another loop.
Ex:
for(int i=1; i<=5; i++)
    for(int j=1; j<=i; j++)
        cout << "Hello World!";

# Loop Control Statements

Loop control statements change execution from its normal sequence.

• break statement :

Terminates the loop or switch statement and transfers execution to the statement immediately following the loop or switch.
Syntax:
break;
Ex:
int n = 1;
while(n <= 100){
    if (n == 50) break;
    cout << "Hello World!";
    n++;
}

• continue statement :

Causes the loop to skip the remainder of its body and immediately retest its condition prior to reiterating.
Syntax:
continue;
Ex:
int n = 1;
while(n <= 100){
    if (n%2 == 0) continue;
    cout << "Hello World!";
    n++;
}

• goto statement :

Transfers control to the labeled statement.
Syntax:
goto label;
labal:
    statement(s);
Ex:
int n;
INPUT:
cout << "Enter Odd Number :";
cin >> n;
if (n%2 == 0) goto INPUT;

# Preprocessor Directives (WIP)

Preprocessor directives or macros are lines included in the code of programs preceded by hash sign (#). These lines are not program statements but directives for the preprocessor. No semicolon (;) is expected at the end of a preprocessor directive.

• Macro Definition

Directive Description
#define To define preprocessor macros #define is used
#undef To undefine macro #undef is used
Syntax:
#define identifier replacement
Ex:
#include<iostream>
using namespace std;

#define PI 3.14159
#define NEWLINE '\n'
#define getMax(a,b) ((a)>(b)?(a):(b))

#undef PI

int main(){
    cout << "Value of Pi = " << PI;
    cout << NEWLINE;

    cout << "Max NO = " << getMax(10,25);

    return 0;
}

• Conditional Directives

Directive Description
#ifdef To define preprocessor macros #define is used
#ifndef To undefine macro #undef is used
#if To undefine macro #undef is used
#endif To undefine macro #undef is used
#else To undefine macro #undef is used
#elif To undefine macro #undef is used
Syntax:
#ifdef 
/* code */
#else
/* code to include if the token is not defined */
#endif
Ex:
#include<iostream>
using namespace std;

#define PI 3.14159
#define NEWLINE '\n'
#define getMax(a,b) ((a)>(b)?(a):(b))

#undef PI

int main(){
    cout << "Value of Pi = " << PI;
    cout << NEWLINE;

    cout << "Max NO = " << getMax(10,25);

    return 0;
}
