# Data Types

• Primitive Built-in Data Types :

Date Type Size Description
bool 1 byte Stores boolean values (true or false)
char 1 byte Stores single character, letter, number or ASCII values
int 4 byte Stores whole numbers, without decimals
float 4 byte Stores fractional numbers (upto 7 decimal digits)
double 8 byte Stores fractional numbers (upto 15 decimal digits)
void - Represents no or absence of type

• Data Type Modifier :

Date Type Size Description
signed 4 byte Store positive, negative integers, and zero
unsigned 4 byte Store only non-negative integer values
short 2 byte Short modifies minimum values that data type can hold
long 8 byte Long modifies maximum values that data type can hold

• Date Type Qualifiers :

Keyword Description
const Used for declaring constant (read-only) variables (cannot be reassigned)
volatile It tells the compiler that variable's value may be changed in ways not explicitly specified by the program
restrict Pointer qualified by restrict is initially the only means by which the object it points to can be accessed 
const int x = 10;
const string name = "Mandar";

• typedef Declarations :

Assign new name (alias) for an existing type using keyword typedef.
Syntax:
typedef type newname;
typedef int feet;
feet distance;

• Enumeration :

An enumeration is user-defined data type that consists of integral constants. To define an enumeration, keyword enum is used.
Syntax:
enum enum_name { list_of_names } var_list;
Ex:
enum week { Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday };

enum color { red, green, blue } c;
c = blue;

# Preprocessor Directives

Preprocessor directives or macros are lines included in the code of programs preceded by hash sign (#). These lines are not program statements but directives for the preprocessor. To define preprocessor macros #define is used. No semicolon (;) is expected at the end of a preprocessor directive. Macros are undefined by using #undef.
Syntax:
#define identifier replacement
#undef identifier
Ex:
#include
using namespace std;

#define PI 3.14159
#define NEWLINE '\n'
#define getMax(a,b) ((a)>(b)?(a):(b))

#undef PI

int main(){
    cout << "Value of Pi = " << PI;
    cout << NEWLINE;

    cout << "Max NO = " << getMax(10,25);

    return 0;
}

Predefined C++ Macros :

Macro Description
__LINE__ This contains current line number of program when it is being compiled
__FILE__ This contains current file name of program when it is being compiled
__DATE__ This contains string of form month/day/year that is date of the translation of source file into object code.
__TIME__ This contains string of form hour:minute:second that is time at which program was compiled.
